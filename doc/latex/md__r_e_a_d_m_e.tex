\#nann

\subsection*{nann Python接口}

在$\ast$$\ast$nann$\ast$$\ast$目录中使用以下命令进行安装 
\begin{DoxyCode}
1 python setup.py build
2 sudo python setup.py install
\end{DoxyCode}
 安装完成后，在\+\_\+python\+\_\+代码中使用{\ttfamily import nann}，即可使用。

20 
21 # 可创建其他的任务结点
22

\end{DoxyCode}
